django-latch
============

Django and Latch integration

Installation
-------------

To install it, simply: ::
   
    git clone https://github.com/rootedcon/django-latch.git
    cd django-latch
    python setup.py install



Requirements
------------

- Python 2.6 or 2.7
- Django >= 1.5

It may work with Python 3 and Django < 1.5 but is not tested
